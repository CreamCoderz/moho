How to build Moho

Moho projects are built using Maven. To build it, simply follow these steps:

  1. Verify that Maven has been installed in your box. you can do this by running the command mvn --version in a terminal. if Maven has been installed, it should print out your installed version of Maven. if not, please install Maven first, refer Maven site.

  2. Now you can run Maven to build Moho projects. Run mvn package to build the project and package the jars.


How to use Moho

Moho is designed to run on any platform that supports JSR 289 and JSR 309. Voxeo Prism is recommended and has Moho included by default.

To run Moho applications on Voxeo Prism, simply deploy any Moho application package to Prism's app directory.

Please note that unless you deploy a custom application router to Voxeo Prism, only one Moho application can be deployed and run at any time.
